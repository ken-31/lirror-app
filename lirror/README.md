# Lirror（リラー）

ふたりの関係を、やさしく映す鏡。
LINEのトーク履歴（.txt）を読み込むだけで、ふたりの関係性を可視化する恋愛コミュニケーション分析アプリ。

**背骨：相手を解読する道具ではなく、自分を客観視する鏡。**
（Lirror_最終設計図_ver2.0.pdf にもとづく v1.0 実装）

## 起動方法

```bash
cd lirror
npm install   # 初回のみ
npm run dev   # → http://localhost:5199
```

本番ビルドは `npm run build`（`dist/` に出力）。

## 機能（v1.0 = 端末内で完結するMVP）

### 💬 LINE分析（ふたりを見る）
- **LINE読み込み** … iOS/Android のトーク履歴 .txt をドロップで解析。サンプルデータでも体験可
- **温度差グラフ** … 「好き度」の一本線は使わず、観測できる行動を二本線で描き、隙間で温度差を語る（7/30/90日切替）
- **吹き出し天秤** … 発信・質問・文長・主導権を1軸統合。重い側が下がる
- **話題マップ** … 話題の割合 × どちらが振ったか ＋ 機能分類（質問/共感/報告/相談/雑談）
- **返信リズム** … 平均返信時間・型（即レス/のんびり/波あり/夜型）・時間帯の重なり帯（★採点はしない）
- **恋愛タイムライン** … 転換点の自動検出 ＋ 手動追記（ラベルは出来事に付ける）
- **今日の天気 / AIひとこと** … 端末内テンプレ生成（送信ゼロ）

### 🌸 あなた（わたしを見る）
- **恋愛プロフィール** … MBTI × ラブタイプ64 の自己入力 → **1024通りのプロフィール**（学習データまとめ.pdf 全収録）
- **「ほしい言葉」タイプ診断** … 6問のシナリオ形式で 肯定/根拠/距離/ユーモア/伴走 の5型を判定
- **今日のお守り** … 型（文体）× 今日の気持ち × LINE実況 の三層で毎日一枚生成。コレクション可

## 対応環境
- **スマホ（〜899px）** … 下タブナビの縦レイアウト。PWAマニフェスト対応（ブラウザの「ホーム画面に追加」でアプリのように起動）
- **パソコン（900px〜）** … 左サイドバー＋2カラムのデスクトップレイアウトに自動で切り替わる

## プライバシー・ファースト
- 全処理を端末内で完結。ネットワーク送信はゼロ
- 保存されるのは集計結果と自己入力のみ（localStorage）。トーク本文は保持しない
- 通知なし。すべて pull 型（開いたときに見に行く）

## 構成（設計図 08章 準拠）

```
src/
  parser/     LINE .txt → Message[]（iOS/Android差を吸収）
  core/       純関数エンジン（非対称性・トレンド・リズム・話題・転換点・天気）
  diagnosis/  「ほしい言葉」型判定 ＋ お守りテンプレマトリクス（5型×3天気）
  data/       ラブタイプ64定義・1024プロフィールJSON（MBTI別に遅延読み込み）
  store/      端末内永続化つきストア
  ui/         画面（ホーム/分析/ふたりの記録/あなた）とSVGチャート
```

コアは純TS関数なので、将来 Expo（react-native-web）へ移す場合も `src/parser` `src/core` `src/diagnosis` `src/data` はそのまま流用できます。
